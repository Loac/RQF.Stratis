***Скрипт раздачи вооружения***

Для подключения скрипта вооружения бойцов по шаблону, необходимо выполнить следующие шаги:
1) Скопировать в папку со свей миссией папку "weapon"
2) Добавить в файл description.ext строчки:
class CfgFunctions{
	class bgr{
		#include "weapon\core\Init.hpp"
	};
};
3) Прописать в строку инициализации бойца например: 
		["EXAMPLE","S1",this] call bgr_fnc_ammoLoad;
4) Подобрать нужный шаблон и тип снаяржения, или создать свой шаблон вооружения.

Для создания своего шаблона вооружения необходимо выполнить следующий шаг:
1) Создать файл шаблона в папке "weapon" по примеру файла EXAMPLE.sqf

ВАЖНО! Пожалуйста, не редактируйте файлы папки "core"!
